Course Description

This course provides an introduction to Computer Science including to the topics of history of the computer, computer architecture, Boolean algebra, binary, and other number systems, and data types, and repersentations. It also teaches the basics of Swift programing, an computer language designed by Apple. The topics include string concantenation, for loops, while loops, and functions.
